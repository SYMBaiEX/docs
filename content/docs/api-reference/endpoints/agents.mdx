## List All Agents

Get a list of all agents with minimal details

### Endpoint

```
GET /api/agents
```

### Response

**Status Code:** 200 OK

Successful response

### Example

<Tabs items={["cURL", "TypeScript", "Python"]}>
  <Tab value="cURL">
    ```bash
    curl -X GET \
      -H "apikey: your-api-key" \
      http://localhost:3000/api/agents
    ```

## Get Agent Logs
