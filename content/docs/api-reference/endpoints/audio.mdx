## Synthesize Audio Message

Synthesize text to audio message

### Endpoint

```
POST /api/audio/{agentId}/audio-messages/synthesize
```

### Parameters

| Name | Type | In | Required | Description |
|------|------|-----|----------|-------------|
| Content-Type | string | header | No | application/json |
| agentId | string | path | Yes | - |

### Request Body

<Tabs items={["Example", "Schema"]}>
  <Tab value="Example">
    ```json
    {
      "text": "Hello, this is a test message",
      "voice": "default"
    }
    ```
  </Tab>
  <Tab value="Schema">
    Content-Type: `*/*`
  </Tab>
</Tabs>

### Response

**Status Code:** 200 OK

Successful response

### Example

<Tabs items={["cURL", "TypeScript", "Python"]}>
  <Tab value="cURL">
    ```bash
    curl -X POST \
      -H "apikey: your-api-key" \
      -H "Content-Type: application/json" \
      -d '{"example": "data"}' \
      http://localhost:3000/api/audio/{agentId}/audio-messages/synthesize
    ```
  </Tab>
  <Tab value="TypeScript">
    ```typescript
    const response = await fetch('http://localhost:3000/api/audio/{agentId}/audio-messages/synthesize', {
      method: 'POST',
      headers: {
        'apikey': 'your-api-key',
        'Content-Type': 'application/json',
      },
      body: JSON.stringify({ example: 'data' }),
    });
    const data = await response.json();
    ```
  </Tab>
  <Tab value="Python">
    ```python
    import requests

    response = requests.post(
        'http://localhost:3000/api/audio/{agentId}/audio-messages/synthesize',
        headers={'apikey': 'your-api-key'},
        json={'example': 'data'}
    )
    data = response.json()
    ```
  </Tab>
</Tabs>
