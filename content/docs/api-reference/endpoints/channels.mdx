## Create Channel Message

Create a new message in a channel

### Endpoint

```
POST /api/messaging/central-channels/{channelId}/messages
```

### Parameters

| Name | Type | In | Required | Description |
|------|------|-----|----------|-------------|
| Content-Type | string | header | No | application/json |
| channelId | string | path | Yes | - |

### Request Body

<Tabs items={["Example", "Schema"]}>
  <Tab value="Example">
    ```json
    {
      "content": "Message content",
      "userId": "user-id",
      "agentId": "{{agentId}}"
    }
    ```
  </Tab>
  <Tab value="Schema">
    Content-Type: `*/*`
  </Tab>
</Tabs>

### Response

**Status Code:** 200 OK

Successful response

### Example

<Tabs items={["cURL", "TypeScript", "Python"]}>
  <Tab value="cURL">
    ```bash
    curl -X POST \
      -H "apikey: your-api-key" \
      -H "Content-Type: application/json" \
      -d '{"example": "data"}' \
      http://localhost:3000/api/messaging/central-channels/{channelId}/messages
    ```
  </Tab>
  <Tab value="TypeScript">
    ```typescript
    const response = await fetch('http://localhost:3000/api/messaging/central-channels/{channelId}/messages', {
      method: 'POST',
      headers: {
        'apikey': 'your-api-key',
        'Content-Type': 'application/json',
      },
      body: JSON.stringify({ example: 'data' }),
    });
    const data = await response.json();
    ```
  </Tab>
  <Tab value="Python">
    ```python
    import requests

    response = requests.post(
        'http://localhost:3000/api/messaging/central-channels/{channelId}/messages',
        headers={'apikey': 'your-api-key'},
        json={'example': 'data'}
    )
    data = response.json()
    ```
  </Tab>
</Tabs>
