Common questions and answers for the Simple Learning Track.

## What is ElizaOS?

ElizaOS is a platform for creating AI agents - think of them as smart chatbots with personality! You
can create agents that interact on platforms like Discord, Twitter, and Telegram without any coding
experience.

## Getting Started

### Q: What are the prerequisites?

A: No coding experience required! You just need:

- Basic understanding of chat applications
- An AI API key (OpenAI, Claude, or Local AI)
- A few minutes to set up

### Q: How much does it cost?

A: Costs range from free to ~$300/month depending on usage:

- Free tier available for testing
- Pay-as-you-go for API usage
- No credit card required to start

### Q: How long does it take to complete?

A: About 1-2 hours total:

- 5 minutes to deploy your first agent
- 15 minutes per platform setup
- 30 minutes for character customization

### Q: What platforms are supported?

A: ElizaOS supports multiple platforms:

- Discord
- Twitter
- Telegram
- Web chat
- And more!

## AI Providers

### Q: Which AI providers can I use?

A: ElizaOS supports multiple AI providers:

- OpenAI (GPT-3.5, GPT-4)
- Claude (Anthropic)
- Local AI models
- Custom providers

### Q: How do I get an API key?

A: Follow these steps:

1. Choose your AI provider
2. Sign up for their service
3. Generate an API key
4. Add it to your `.env` file

## Agent Creation

### Q: How do I create a custom agent personality?

A: You can customize your agent by:

1. Editing the character file
2. Setting name, description, and personality traits
3. Adding custom knowledge and responses
4. Testing and refining

### Q: Can I use templates?

A: Yes! Templates are available for:

- Different platforms
- Various use cases
- Pre-built personalities
- Common configurations
