## Prerequisites

Before you begin, ensure you have:

<Steps>
  <Step>
    **Node.js** (v23.3.0 or higher)
    
    Download from [nodejs.org](https://nodejs.org/) or use a version manager like [nvm](https://github.com/nvm-sh/nvm).
  </Step>
  <Step>
    **Bun** (recommended package manager)
    
    Install from [bun.sh](https://bun.sh/docs/installation)
  </Step>
  <Step>
    **API Keys** for your chosen model provider
    
    - OpenAI API key (recommended for beginners)
    - Anthropic API key (for Claude models)
    - Or set up local models with Ollama
  </Step>
  <Step>
    **Windows Users**: WSL 2 required
    
    Install [WSL 2](https://learn.microsoft.com/en-us/windows/wsl/install-manual) for Windows compatibility.
  </Step>
</Steps>

## Installation Methods

ElizaOS offers multiple installation approaches:

### Using the CLI (Recommended)

The ElizaOS CLI is the fastest and most reliable way to get started:

```bash
# Install the CLI globally
bun install -g @elizaos/cli

# Create your first project
elizaos create my-agent
```

### Manual Installation

For developers who prefer manual control:

```bash
# Clone the repository
git clone https://github.com/elizaOS/eliza.git
cd eliza

# Install dependencies
bun install
bun run build
```

### Docker Setup

Run ElizaOS in a containerized environment:

```bash
# Using Docker Compose
docker-compose up -d
```

### Quick Fixes

- **Node.js version issues**: Ensure you're using Node.js v23.3.0+
- **Permission errors**: Try using `sudo` or check your Node.js installation
- **API key errors**: Verify your API keys are correctly set in the `.env` file
- **Port conflicts**: ElizaOS uses port 3000 by default - ensure it's available
